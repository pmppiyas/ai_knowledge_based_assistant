## Overview

AI Knowledge Base Assistant is a modern RAG-based AI system that allows users to:

- Upload PDFs and documents
- Generate embeddings
- Store vectors in a vector database
- Retrieve relevant knowledge
- Chat with documents using AI

This project is designed as a scalable backend foundation for building:

- AI SaaS platforms
- Document chatbots
- AI research assistants
- Internal company knowledge systems
- Multi-document AI search engines
